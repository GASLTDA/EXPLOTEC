Mass Cancel Invoice(s)

This Module allows you to cancel multiple Invoices at a time.


How To Use


Once module is installed you can see 'Mass Cancel Invoice(s)' in a action window which will appear only after selecting any Invoice.


Mass Cancel Invoice(s) in action window

Giving permission to cancel all selected Invoices.

After giving permission, click on 'Cancel Invoice' to cancel all selected Invoice.


If permission is not given then warning is raise informing to check the check-box.

Below you can see the warning if the checkbox is not clicked.


On clicking Cancel Button after giving permission from the wizard, Invoices will get cancelled.

You can see in below image that multiple invoices get cancelled.


If any of the selected invoice which cannot be cancel then it will give warning message with Invoice name saying you cannot cancel this Invoice.
